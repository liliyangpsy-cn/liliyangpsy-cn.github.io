# 网站设计风格指南

## 设计理念

### 色彩方案
- **主色调**：深海蓝 (#1e3a8a) - 体现学术严谨性和专业深度
- **辅助色**：温暖米色 (#f5f5dc) - 呼应苗族文化中的温暖色调
- **强调色**：琥珀金 (#f59e0b) - 突出重要信息和成就
- **文本色**：炭灰色 (#374151) - 确保良好的可读性

### 字体设计
- **标题字体**：Playfair Display - 优雅的衬线字体，体现学术气质
- **正文字体**：Inter - 现代无衬线字体，确保清晰易读
- **代码字体**：JetBrains Mono - 用于技术内容展示

### 视觉语言
- **学术现代主义**：简洁、专业、有层次感
- **文化融合**：融入苗族传统图案元素作为装饰
- **科技感**：通过动画和交互体现AI研究背景

## 视觉效果

### 使用的核心库
1. **Anime.js** - 页面元素动画和过渡效果
2. **ECharts.js** - 学术数据可视化
3. **p5.js** - 创意背景效果和粒子系统
4. **Splide.js** - 图片轮播和项目展示
5. **Leaflet** - 学术经历地图展示
6. **Matter.js** - 物理动画效果
7. **Pixi.js** - 高性能视觉特效

### 动画效果
- **文字动画**：逐字符显示、颜色循环强调
- **图片效果**：Ken Burns平移缩放、无限滚动
- **背景效果**：粒子系统、流体渐变动画
- **交互反馈**：悬停3D倾斜、阴影扩展

### 布局特色
- **网格系统**：基于Tailwind CSS的响应式布局
- **卡片设计**：圆角、阴影、层次分明
- **视觉层次**：通过大小、颜色、间距建立信息层级

### 头部效果
- **英雄区域**：全屏背景图片配合渐变遮罩
- **动态标题**：打字机效果配合颜色变化
- **导航栏**：半透明背景，滚动时动态变化
