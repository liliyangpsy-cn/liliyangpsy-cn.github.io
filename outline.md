# 个人网站项目概述

## 项目目标
为申请美国咨询心理学博士项目创建专业个人网站，展示学术背景、研究经历和临床实践经验。

## 网站结构

### 1. 首页 (index.html)
- **英雄区域**：个人简介和核心优势
- **研究概览**：主要研究方向和成果
- **成就亮点**：重要奖项和发表
- **技能展示**：交互式技能雷达图

### 2. 研究经历 (research.html)
- **时间线展示**：研究项目发展历程
- **项目详情**：每个研究项目的详细介绍
- **方法论**：使用的研究方法和工具
- **成果展示**：论文、报告、演示

### 3. 个人陈述 (statement.html)
- **学术理念**：研究动机和价值观
- **个人背景**：文化背景对学术的影响
- **未来规划**：博士阶段研究计划
- **职业目标**：长期职业发展愿景

### 4. 联系方式 (contact.html)
- **联系信息**：邮箱、电话、地址
- **社交媒体**：学术社交网络
- **预约系统**：会议时间预约
- **快速联系**：在线联系表单

## 技术实现

### 前端技术栈
- **HTML5**：语义化标记
- **Tailwind CSS**：响应式样式框架
- **JavaScript ES6+**：现代JavaScript特性
- **核心库**：Anime.js, ECharts.js, p5.js, Leaflet等

### 文件结构
```
/output/
├── index.html              # 主页
├── research.html           # 研究经历页面
├── statement.html          # 个人陈述页面
├── contact.html            # 联系方式页面
├── main.js                 # 主要JavaScript逻辑
├── resources/              # 资源文件夹
│   ├── hero-bg.jpg        # 英雄区域背景图
│   ├── profile.jpg        # 个人照片
│   ├── research-*.jpg     # 研究项目图片
│   └── awards-*.jpg       # 奖项证书图片
└── README.md              # 网站管理指南
```

## 内容策略

### 学术重点
- 突出AI辅助心理治疗研究
- 强调多元文化背景和敏感性
- 展示临床实践经验
- 体现跨学科合作能力

### 个人品牌
- 专业而不失人文关怀
- 科技感与文化认同并重
- 学术严谨性与创新思维结合
- 国际化视野与本土实践融合
