# 个人网站管理指南

## 网站概述

这是一个为申请美国咨询心理学博士项目而创建的专业个人网站，展示学术背景、研究经历和临床实践经验。

### 网站结构
- **首页 (index.html)**: 个人简介、核心优势、技能展示
- **研究经历 (research.html)**: 详细项目展示、交互式时间线
- **个人陈述 (statement.html)**: 学术理念、个人背景、未来规划
- **联系方式 (contact.html)**: 多种联系方式、智能推荐系统

## 技术特性

### 前端技术栈
- **HTML5**: 语义化标记
- **Tailwind CSS**: 响应式样式框架
- **JavaScript ES6+**: 现代JavaScript特性
- **核心库**:
  - Anime.js: 页面元素动画
  - ECharts.js: 数据可视化
  - p5.js: 创意背景效果
  - Leaflet: 交互式地图
  - Splide.js: 图片轮播

### 设计特色
- **响应式设计**: 适配桌面、平板、手机
- **现代学术风格**: 专业而不失温暖
- **文化元素**: 融入苗族传统图案
- **交互体验**: 丰富的动画和交互效果

## 内容管理

### 更新个人信息
1. **基本信息修改**
   - 编辑 `index.html` 中的个人信息
   - 更新联系方式在 `contact.html`
   - 修改个人照片在 `resources/profile.jpg`

2. **研究项目更新**
   - 编辑 `research.html` 中的项目卡片
   - 添加新的研究成果和论文
   - 更新项目时间线和地图标记

3. **内容格式化**
   - 使用语义化HTML标签
   - 保持一致的样式类名
   - 确保图片尺寸和格式统一

### 图片资源管理
- **英雄背景**: `resources/hero-bg.jpg` (1920x1080)
- **个人照片**: `resources/profile.jpg` (500x500)
- **研究配图**: `resources/ai-research.jpg` 等
- **图片优化**: 建议使用WebP格式，压缩至200KB以下

## 功能定制

### 交互功能
1. **技能雷达图**: 在 `main.js` 中修改 `initSkillsChart()` 函数
2. **研究筛选**: 在 `research.html` 中添加新的分类标签
3. **联系表单**: 集成邮件服务API或邮件转发

### 视觉效果
1. **粒子背景**: 调整 `main.js` 中的粒子参数
2. **动画效果**: 修改Anime.js动画配置
3. **颜色主题**: 更新CSS变量值

## SEO优化

### 元标签优化

## 故障排除

### 常见问题

3. **表单提交失败**: 验证服务器配置
4. **布局错乱**: 检查CSS类名和结构
